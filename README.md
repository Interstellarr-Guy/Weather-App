Weather-App (React + Node.js + weatherAPI)

A simple full stack Weather App built using React (frontend) and Node.js +EXpress (backend).
The app fetches real-time weather data from WeatherAPI,while keeping the API key secure on the server.

Features 
* Search weather by city name
* Shows temperature and condition
* API key hidden using backend & environment variables
* Full-stack setup (Frontend + Backend )

Tech Stack
   Frontend
   * React
   * Axios
   * CSS

 Backend
 * Node.js
 * Express.js
 * Axios
 * dotenv
 * cors

External API
* WeatherAPI

API Flow
React App --> Node/Express Backend -->WeatherAPI
